# Przychodnia_szkielet
Zarys aplikacji, która posłuży do rozwoju aplikacji na Androida.

Aplikacja ma zasymulować system, który umożliwia obsługę pacjentów przez personel medyczny. 
Zakres działania obecnie obejmować będzie dodawnie/usuwanie pacjentów, rejestracja/rezygnacja na/z wizyty, zmiana danych pacjenta,
sprawdzenie listy pacjentów, sprawdzenie listy lekarzy oraz ich pacjentów (wizyty). W przyszłości być może pojawią się inne fukcnje. 
Jednak aplikacja w c++ ma jedynie nakreślić szkielet.

Zaimplementowanie podstawowych funkcjonalności, w przyszłości rozwój a ostatecznie przeniesienie się na platformę android.
Prawdopodobne przechowywanie modeli w bazie danych.
